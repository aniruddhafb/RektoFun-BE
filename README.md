⚔️ RektoFun Backend

Backend infrastructure powering Rekto.fun — the first PvP battleground for predictions built on Solana.


This repository contains the backend services responsible for:

Challenge management

Prediction validation

User & clan systems

Escrow & challenge resolution logic

Sports event verification

Leaderboards & REKT points systems
